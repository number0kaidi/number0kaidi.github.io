#5.两种虚拟机的不同 (熟练了解) 

jvm : java 虚拟机   sun
(dvm) : dalvik虚拟机   google


版权问题。

jvm:执行的是多个.class类型的文件；基于栈架构；栈是内存一个空间，执行指令需要向CPU寻址后才会执行；
dvm:执行的是一个.dex类型的文件； 基于寄存器架构；寄存器是CPU的一个组成部分；执行执行和操作无需寻址，可以直接执行；

#6.art 模式 android runtime 了解
空间换时间；
art模式：应用程序在安装时进行预编译读取，将程序转换成机器码存储；以后再次运行程序，就直接加载机器码运行；无需每次运行编译；所以运行程序的速度快一些；但是，由于进行预编译读取，所以安装时间会 稍长一些，并且会占用一些存储空间；

#14.ADB指令练习 (重点) 掌握

adb 运行的端口号是5037

adb android debug bridge 安装调试桥；  建立手机与电脑之间的链接；

配置adb的环境变量；C:\kaifa\adt-bundle-windows-x86_64_20140101\sdk\platform-tools


adb  push  pc_filePath  phone_filePath ;//将电脑文件放到手机中
adb pull phone_filePath  pc_filePath;//将手机文件拉到电脑上
adb install  -r  apkPath  ; 安装应用； -r强制安装；
adb uninstall  packagename; 卸载应用

adb kill-server:关闭adb的链接
adb start-server:开启adb的链接
netstat -oan : 查看端口占用情况

adb shell:进入手机linux环境下
adb  devices：查看当前电脑所连接的android设备；
adb shell + logcat 可以查看手机运行时的日志；

注意：如果当前电脑链接多台android设备，使用adb操作时，需要加-s ，指定操作的是哪台设备；

#16.按钮的点击事件的4种写法 掌握

1.匿名内部类的方式 
2.写个类实现OnclickListener ； setOnclicklistener时传一个类的对象啊
3.让当前类实现OnclickListener ； setOnclicklistener时传this； 一般公司开发中用这个；
4.在布局文件中为当前控件添加一个Onclick属性，在对应的Activity中创建一个以该属性值为名方法；要public，名字对应，view参数；底层利用反射技术；一般公司不用；主要用于简单的测试.
